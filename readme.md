# 冰兔的工具站

一个基于 Jekyll 构建的静态工具网站，提供各种实用的在线工具，所有操作均在浏览器本地完成，保护用户隐私。

## 🌟 项目简介

冰兔的工具站是一个完全在前端运行的工具集合网站，用户无需担心数据上传到服务器的问题。所有工具都在本地浏览器中运行，确保数据安全和隐私保护。

### 主要特色

- 🔒 **完全本地运行** - 所有数据处理都在浏览器中完成
- 🛠️ **丰富工具集合** - 涵盖图像处理、编码解码、文本转换等多个领域
- 📱 **响应式设计** - 支持桌面端和移动端访问
- 🎨 **现代化界面** - 简洁美观的用户界面设计
- ⚡ **无需安装** - 直接通过浏览器访问使用

## 🏗️ 项目架构

### 技术栈

- **前端框架**: Jekyll (静态网站生成器)
- **样式**: CSS3 + 响应式设计
- **脚本**: 原生 JavaScript
- **部署**: 静态文件托管

│   ├── text2png.html            # 文字转图片工具
│   └── video2img.html           # 视频帧提取工具

│   ├── style.css       # 主样式文件
│   └── tool_common.css # 工具页面通用样式

### 核心组件

#### 1. 布局系统 (`_layouts/default.html`)
- 统一的页面布局模板
- 包含页眉、侧边栏导航、主内容区域和页脚
- 响应式设计支持

#### 2. 样式系统
- **`style.css`**: 主要样式文件，包含全局样式、导航栏样式、响应式设计
- **`tool_common.css`**: 工具页面通用样式，包含按钮、表单、加载动画等组件样式

#### 3. 工具架构
每个工具页面都遵循统一的架构：
- 使用 Jekyll Front Matter 定义页面元数据
- 集成防重复点击控制器
- 统一的加载状态管理
- 响应式布局支持

#### 4. 导航系统
- 侧边栏导航，支持多级菜单
- 移动端汉堡菜单
- 当前页面高亮显示

## 🛠️ 工具分类

### 图像处理工具
- **盲水印工具** - 基于DWT小波域的数字水印嵌入与提取
- **幻影坦克生成器** - 创建在不同背景下显示不同图像的图片
- **低质量JPEG生成器** - 故意降低图片质量用于特定用途
- **视频帧提取器** - 从视频文件中提取帧图像

### 编码解码工具
- **摩斯电码工具** - 文本与摩斯电码的相互转换
- **二维码生成器** - 生成各种类型的二维码
- **二维码隐藏工具** - 在图片中隐藏二维码信息

### 文本处理工具
- **文字转图片工具** - 将文本转换为图片格式

### 文件处理工具
- **图片ZIP伪装工具** - 将ZIP文件伪装成图片格式

## 🚀 快速开始

### 本地开发

1. **安装 Jekyll**
   ```bash
   gem install jekyll bundler
   ```

2. **安装依赖**
   ```bash
   bundle install
   ```

3. **启动本地服务器**
   ```bash
   bundle exec jekyll serve --host 0.0.0.0 --port 8000 --livereload
   ```

4. **访问网站**
   打开浏览器访问 `http://localhost:8000/tools-site/`

### 添加新工具

1. 在 `tools/` 目录下创建新的 HTML 文件
2. 使用 Jekyll Front Matter 定义页面信息：
   ```yaml
   ---
   layout: default
   title: 工具名称 - 冰兔的工具站
   header_title: 工具标题
   header_description: 工具描述
   ---
   ```
3. 在侧边栏导航中添加链接
4. 编写工具的核心 JavaScript 功能
